options（Object）：参数对象

}

mapConfig：

Example

const MapConfig={

Events

地图事件

click

map.on('click', function(ev) {

    

    
    alert(ev.latlng); // ev is an event object (MouseEvent in this case)

    
});

contextmenu

当用户在地图上按下鼠标右键时激发，会阻止默认浏览器上下文菜单显示此事件是否有侦听器。当用户按住一秒钟（也称为长按）时，也会在手机上触发。

preclick

在鼠标单击地图之前激发（当您希望在任何现有单击处理程序开始运行之前单击时发生某些事情时，有时会很有用）

keypress

当用户在地图聚焦时按下键盘上的键时激发。

图层事件:

baselayerchange，overlayadd，overlayremove，layeradd，layerremove

地图状态改变事件:

load,upload,move,zoom,zoomed,moveed,resize等

弹出框事件:

popupopen,popupclose，autopanstart

Methods

getRenderer(Path)

返回应用于渲染给定路径的渲染器实例。它将确保尊重贴图和路径的渲染器选项，并且贴图上确实存在渲染器。

L.GW.Layer

图层

L.GW.Layer.LayerGroup(map)

Methods

toGeoJSON()

返回图层组的GeoJSON表示形式（如：GeoJSON FeatureCollection、GeometryCollection或MultiPoint）。

addLayer()

将给定图层添加到组中

removeLayer(Layer/Number)

根据图层自身或_leaflet_id移除图层

hasLayer(Layer/Number)

根据图层自身或_leaflet_id判定是否已添加到组中，是则返回true

clearLayers()

清除所有图层

invoke(methodName,methodName……)

在此组中包含的每个层上调用methodName，并传递任何其他参数。如果包含的层未实现methodName，则无效。

eachLayer(function(){})

在组的各个层上迭代，可以选择指定迭代器函数的上下文。

group.eachLayer(function (layer) {

    

    
    layer.bindPopup('Hello');

    
});

getLayer(id)

id为Number。返回具有给定内部ID的图层

getLayers()

返回组中的所有图层的数组

setZIndex()

在此组中包含的每个层上调用setZIndex，并传递z索引

getLayerId(Layer)

返回层的内部ID

清除所有图层

setStyle(Path options)

为具有setStyle方法的组的每个层设置给定的路径选项。

layer.setStyle({

    

    
                weight: 2,

    
                opacity: 1,

                fillColor: '#FFEDA0'})

bringToFront()

将图层组置于所有其他图层的顶部

bringToBack()

将图层组带到所有其他图层的后面

getBounds()

返回要素编组的LatLngBounds（根据其子要素的边界和坐标创建）

L.GW.Geom

几何要素

L.GW.Geom.Point

点位坐标，不可单独使用。现有类型有PixelPoint（屏幕像素点）和GeomPoint（空间几何点）.

L.GW.Geom.Point(options)

Example

xxxxxxxxxx

    

    

    

    

        

            

            

            
const options={

        

        
  type:'GeomPoint',

        
  x:116,

        
  y:40

        
}

        
const GeomPoint= new L.GW.Geom.Point(options)

        
//具体用法

        
map.panTo(GeomPoint) //以该点为地理中心

        
//当type为PixelPoint时，map.panBy(point)

        
const Pixeloptions={

        
        type:'PixelPoint',

        
        x:200,

        
        y:300

        
      }

        
const Pixelpoint= new L.GW.Geom.Point(Pixeloptions);

Methods

GeomPoint：

equals(point)

俩点相等返回true，不相等返回false

toString（）

返回点的字符串，如：GeomPoint.toString(),返回'LatLng(40, 116)'

distanceTo(point)

计算到另一个点的距离,返回类型Number，单位：米

wrap()

当地图轮番循环时，返回该点的一个新LatLng对象，如Geompoint.wrap()

toBounds(Number)

返回以目标点为中心，半径为(Number/2)米的矩形坐标。

如：

xxxxxxxxxx

    

    

    

    

        

            

            

            
Geompoint.toBounds(100)

        

        
//返回：

        
{

        
_northEast:{lat: 40.000449157638535, lng: 116.0005863336554},

        
_southWest: {lat: 39.999550842361465, lng: 115.9994136663446}

        
}

clone()

复制点，Geompoint.clone()

PixelPoint:

add(Pixelpoint)

返回当前点与给定点相加的结果。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.add(Pixelpoint);

        

        
//返回结果：

        
{x: 400, y: 600}

clone()

返回当前点的副本，Pixelpoint.clone()

contains(Pixelpoint)

如果给定点的两个坐标小于相应的当前点坐标（绝对值），则返回true。

xxxxxxxxxx

    

    

    

    

        

            

            

            
//Pixelpoint.contains({x:100,y:400})//返回false

        

        
Pixelpoint.contains({x:200,y:400}) //返回true

toString()

将当前点字符串表示形式，Pixelpoint.toString()，返回'Point(200.1, 300.6)'

subtract(Pixelpoint)

返回从当前值减去给定点的结果。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.subtract({x:100,y:200}) //返回{x:100,y:100}

divideBy(Number)

返回当前点除以给定数字的结果。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.divideBy(5)//返回{x:40,y:60}

multiplyBy(Number)

返回当前点乘以给定数字的结果。

scaleBy(Pixelpoint)

将当前点的每个坐标乘以比例的每个坐标。在线性代数术语中，将点乘以由比例定义的比例矩阵。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.scaleBy({x:10,y:20})//返回{x:2000,y:6000}

unscaleBy(Pixelpoint)

将当前点的每个坐标除以比例的每个坐标。在线性代数术语中，将点除以由比例定义的比例矩阵。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.unscaleBy({x:10,y:20})//返回{x:20,y:15}

ceil()

返回当前点向上取整的点

xxxxxxxxxx

    

    

    

    

        

            

            

            
const Pixelpoint.x= 200.2, Pixelpoint.y= 300.6;

        

        
Pixelpoint.ceil() //返回{x: 201, y: 301}

round()

返回具有四舍五入坐标的当前点的副本

floor()

返回当前点向下取整的点

trunc()

返回具有截断坐标（向零舍入）的当前点的副本。

distanceTo(Pixelpoint)

返回当前点和给定点之间的笛卡尔距离,单位屏幕像素。

equals(Pixelpoint)

如果给定点具有相同的坐标，则返回true。

xxxxxxxxxx

    

    

    

    

        

            

            

            
Pixelpoint.equals({x:200.1,y:300.6})//true

        

        

            
Pixelpoint.equals({x:200,y:300.6})//false
